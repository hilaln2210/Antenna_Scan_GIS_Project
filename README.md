# Antenna Scan GIS Project

מיפוי אנטנות סלולריות בישראל — פרויקט GIS לאיתור ומיקום אנטנות. ריכוז מידע ממקורות שונים בממשק ידידותי.

## צילומי מסך

### דף בית

![דף בית](screenshots/home.png)

## תכונות

- **דף בית** — כניסה לאפליקציה
- **Login / Register** — התחברות והרשמה
- **Map** — מפת אנטנות אינטראקטיבית
- **Rating and Opinion** — דירוג ומשוב על אנטנות

## טכנולוגיות

- **HTML** · **CSS** · **JavaScript** · **GIS/מפות**

## הרצה

```bash
# שרת מקומי
python3 -m http.server 8000
# פתחי בדפדפן: http://localhost:8000/Home.html
```

או פתחי ישירות `Home.html` בדפדפן.

|------|-------|
